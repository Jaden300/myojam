myojam — Installation

Requirements: macOS 12+, Python 3.13

1. Install dependencies:
   /Library/Frameworks/Python.framework/Versions/3.13/bin/pip3 install PyQt6 scipy scikit-learn numpy pyserial

2. Install cliclick:
   brew install cliclick

3. Run:
   chmod +x run.sh && ./run.sh

4. Grant Accessibility permission when prompted.
